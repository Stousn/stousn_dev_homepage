Das Projekt

Als 2018 die TLD .dev released wurde, wollte ich unbedingt eine haben. Das Problem war ursprünglich der Inhalt. Ich habe im ersten Schritt meine eigentliche .at-Homepage kopiert. Während dem Schreiben meiner Masterarbeit, kam ich auf die Idee, Projekte hier darzustellen. Inzwischen nutze ich sie als Homepage für meine Freiberuflichkeit.

Technologien

Zuerst war es eine 1:1 Kopie meiner Homepage mit unterschiedlichem Content. Im Sommer 2019 habe ich die Hauptseite angepasst und dort README.md-Files von Github drgestellt. Es kam ein Framework zum parsen von Markdown zu HTML dazu, um die Inhalte schön darzustellen. Dann habe ich die Seite technisch komplett neu geschrieben und nutzte dafür ein JavaScript Framwork um sie als Single-Page-Applikation umzusetzen: a vanillajs-spa von rishavs hat sich als praktisch erwiesen und ich habe es etwas angepasst. Auch die Logik zum Anzeigen von README.md Files ist wurde wieder integriert.
Da ich aber 2023 überlegt habe, ein Kontaktformular zu integrieren, habe ich mich dazu entschieden die Seite nochmal umzustellen. Diesmal auf Angular.
