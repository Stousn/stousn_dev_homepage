Persönlicher Linkbaum

Die gängigsten Social Media Profile haben nur die Möglichkeit einen Link zu hinterlegen. Willst du rechtlich sauber unterwegs sein, musst du diesen Link für das Impressum verwenden. Möchtest du doch lieber andere Social Media Kanäle oder ähnliches verlinken, dann benötigst du eine Webseite mit einem Link-Baum. Dafür gibt es zahlreiche Anbieter (teilweise sogar kostenlos). Das Problem ist meist, dass diese Seiten aber Mittracken und die Seiten zumeist in den USA gehostet werden, was wiederum einen DSGVO-Verstoß darstellt. Gerade für Influencer und Kleinunternehmer kann das zum Problem werden.

Die Lösung ist dein eigener Link-Baum. Farblich nach deinen Vorstellungen gestaltet, rechtlich absolut DSGVO-konform und nebenbei ein Aushängeschild als Mini-Homepage.

Das exklusive Angebot: Eigene Link-Baum-Seite für dein soziales Netzwerk. Auf wunsch mit eigener Domain. Inklusive 50 Link-Änderungen pro Jahr.
