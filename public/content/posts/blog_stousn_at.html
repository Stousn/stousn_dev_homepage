The development of my blog blog.stousn.at

The Project

I had blogs earlier during school time and my full time study. However, during part time studying I had no time for it. But now, as I finished at FH JOANNEUM, I created a new blog to share my photographs, ideas, thoughts and videos.

Technologies

I wanted to write my blog-posts very simple with Markdown, because I use Markdown for nearly any text i write. So I searched for solutions and found Hugo. A software written in Go that creates static HTML pages from Markdown. I host the blog on GitHub pages.

Status

The blog is live and you view it at blog.stousn.at. Currently, I just need to write more/frequent posts.
